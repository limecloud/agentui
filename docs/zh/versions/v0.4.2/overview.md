# Agent UI v0.4.2

Agent UI v0.4.2 增加 Agent 标准家族的公开曝光入口。该版本从 documentation site、LLM entrypoints 与 version navigation 中互链 Agent Knowledge、Agent UI、Agent Runtime 与 Agent Evidence。

## Highlights

- 增加 [Agent 标准生态](../../reference/agent-ecosystem.md) reference page。
- 增加 Agent Knowledge、Agent UI、Agent Runtime 与 Agent Evidence 的 mutual links。
- 增加 Agent Artifact、Agent Tool、Agent Policy、Agent Context、Agent Evaluation、Agent Workflow、Agent Model Routing 等未来标准候选。
- 更新 `llms.txt` / `llms-full.txt`，方便 AI clients 发现标准家族。
- Agent UI core specification 与 v0.4.1 保持兼容。
