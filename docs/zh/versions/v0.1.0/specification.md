# Agent UI v0.1.0 规范

v0.1.0 的规范快照与当前 [latest 规范](/zh/specification) 对齐。

## v0.1.0 变化

- Agent UI 被引入为 Agent Skills 和 Agent Knowledge 的互操作伙伴标准。
- `AGENTUI.md` 被定义为必需包入口。
- 定义标准目录：`patterns/`、`surfaces/`、`contracts/`、`states/`、`examples/`、`schemas/`、`evals` 和 `assets/`。
- 定义标准表面：Conversation、Process、Task、Artifact、Evidence。
- Runtime projection 规则要求 UI state 与 runtime facts 分离。
- 定义 catalog、guide、surface、contract、example 的渐进加载层级。

完整字段、目录和运行时规则见 [latest 规范](/zh/specification)。
