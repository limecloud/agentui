# v0.4.7 概览

Agent UI v0.4.7 是标准生态发现 patch release。它把 Agent Tool 提升为当前标准，并刷新 README、版本导航和 LLM 入口。

- 面向 AI clients 的 LLM entrypoint refresh。
- 不改变 Agent UI 的核心协议语义。
