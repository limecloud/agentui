# Agent UI v0.4.3

Agent UI v0.4.3 修复并打磨本地化文档首页，让 `/en/` 与 `/zh/` 都成为面向读者和 AI 客户端的正式 landing page。

## 重点

- 将本地化 index 页面修正为真正的 `layout: home` landing page。
- 为规范、快速开始、标准生态与 LLM 上下文提供更清晰的 hero actions。
- 首页保持简洁，把详细概念交给专门文档页承载。
- 修复 GitHub Pages repository-base 部署下的站点 logo 路径。
- 核心 Agent UI 规范继续兼容 v0.4.2。
