# v0.4.3 规范快照

v0.4.3 是文档首页修复版本。核心 Agent UI 规范继续兼容 v0.4.2。

## 首页契约

兼容文档 SHOULD 将本地化首页作为 landing page 暴露：

- `/en/` 与 `/zh/` index routes 使用 `layout: home` frontmatter。
- 明确提供规范、快速开始、标准生态与 LLM 上下文 actions。
- 首页只保留简洁标准归属和 start-here links。
- 不把 locale root 渲染成带侧栏的普通概览文档页。
