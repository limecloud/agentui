# v0.4.5 概览

Agent UI v0.4.5 是标准生态发现补丁版本。它把 Agent Policy 提升到当前 standards map，并刷新 README、版本导航和 LLM entrypoints。

## Included

- 当前标准表加入 Agent Policy 链接。
- boundary map 增加 policy decision layer。
- 刷新 AI clients 使用的 LLM entrypoints。
- 不改变 Agent UI 核心协议语义。
