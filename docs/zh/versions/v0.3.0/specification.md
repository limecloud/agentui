# Agent UI v0.3.0 规范

v0.3.0 的规范快照与当前 [latest 规范](/zh/specification) 对齐。

## v0.3.0 变化

- 将 Agent UI 重新表述为生态级投影标准，而不是某一两个相邻标准的狭义补充。
- 新增 runtime、model、tools、skills/workflows、memory/knowledge/context stores、artifacts、evidence、permissions、sessions 和 design systems 的生态边界说明。
- 将边界说明移动到 [生态边界](/zh/reference/ecosystem-boundaries)。
- 删除会暗示错误范围的旧边界页源文件。
- 保持 v0.2 的 runtime-first 表面模型和事件投影指导不变。

完整运行时和表面规则见 [latest 规范](/zh/specification)。
