# v0.2.0 Changelog

## Added

- Runtime-first specification.
- Composer surface standard.
- Message Parts surface standard.
- Runtime Status surface standard.
- Tool UI surface standard.
- Task Capsule surface standard.
- Human-in-the-loop surface standard.
- Artifact/Canvas surface standard.
- Timeline/Evidence surface standard.
- Session/Tabs surface standard.
- Runtime event projection contract.
- Backend coordination contract.
- Performance metrics contract.
- Session hydration implementation guide.
- Queue and steer implementation guide.
- Acceptance scenarios guide.
- Research sources reference.
- Mermaid rendering support in the documentation site.
- v0.2.0 version snapshots.

## Changed

- Agent UI is now presented as a runtime projection contract.
- Navigation now exposes Surfaces, Contracts, Implementation, and Research as first-class sections.
- Quickstart now starts from an event adapter and projection store.
- Project version is aligned to `0.2.0`.
- Release notes now describe runtime projection, backend coordination, performance contracts, and legacy compatibility.

## Compatibility

- v0.1.0 wording is superseded.
- New implementations should start from runtime events and snapshots.
