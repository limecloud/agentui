# Agent UI v0.3.0 Specification

The v0.3.0 specification snapshot is aligned with the current [latest specification](/en/specification).

## v0.3.0 changes

- Reframes Agent UI as an ecosystem-level projection standard, not a narrow companion to one or two adjacent standards.
- Adds ecosystem boundary guidance for runtimes, models, tools, skills/workflows, memory/knowledge/context stores, artifacts, evidence, permissions, sessions, and design systems.
- Moves the boundary guide to [Ecosystem boundaries](/en/reference/ecosystem-boundaries).
- Removes the old boundary page source files that implied the wrong scope.
- Keeps the v0.2 runtime-first surface model and event projection guidance intact.

See the [latest specification](/en/specification) for the full runtime and surface rules.
