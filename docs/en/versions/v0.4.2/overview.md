# Agent UI v0.4.2

Agent UI v0.4.2 adds public ecosystem exposure for the Agent standards family. The release connects Agent Knowledge, Agent UI, Agent Runtime, and Agent Evidence from the documentation site, LLM entrypoints, and version navigation.

## Highlights

- Adds the [Agent standards ecosystem](../../reference/agent-ecosystem.md) reference page.
- Adds mutual links to Agent Knowledge, Agent UI, Agent Runtime, and Agent Evidence.
- Adds future standard candidates such as Agent Artifact, Agent Tool, Agent Policy, Agent Context, Agent Evaluation, Agent Workflow, and Agent Model Routing.
- Updates `llms.txt` / `llms-full.txt` so AI clients can discover the standards family.
- Keeps the Agent UI core specification compatible with v0.4.1.
