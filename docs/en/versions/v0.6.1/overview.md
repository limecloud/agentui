# Agent UI v0.6.1

Agent UI v0.6.1 is a patch release that aligns UI projection with the Agent Runtime profile without moving runtime ownership into the UI standard.

## Highlights

- Adds runtime profile projection test cases for runtime ids, read model projection, tool approval, task retry, routing, evidence, replay, review, hydration, and governance failures.
- Aligns the runtime event projection contract with Agent Runtime-style events, thread read models, task snapshots, and evidence packs.
- Adds optional runtime profile correlation fields to the public event schema.
- Refreshes README, navigation, LLM entrypoints, release notes, package metadata, and version snapshots for v0.6.1.
