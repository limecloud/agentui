# Agent UI v0.6.1

Agent UI v0.6.1 is a runtime-profile projection release. It makes Agent Runtime events, read models, task snapshots, and evidence packs explicitly testable from UI projection without making UI the source of truth.

## Highlights

- Adds English and Simplified Chinese runtime profile projection test cases for runtime ids, read model projection, tool approval, task retry, routing, evidence, replay, review, hydration, and governance failures.
- Aligns the runtime event projection contract with Agent Runtime-style `RuntimeEvent`, `ThreadReadModel`, `TaskSnapshot`, and `EvidencePack` sources.
- Extends the public Agent UI event schema with optional runtime profile correlation fields.
- Refreshes package metadata, navigation, version snapshots, README, and LLM entrypoints to v0.6.1.
